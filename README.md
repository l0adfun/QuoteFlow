# QuoteFlow

**QuoteFlow** is a sleek and minimal web application that displays random inspirational quotes with a dynamic reading timer. Users can pause/resume the timer and copy quotes easily. The reading duration is automatically calculated based on the length of each quote for an optimal reading experience.

---

## Features

- Fetches random quotes from [Quotable API](https://api.quotable.io).  
- Dynamic reading timer based on quote length.  
- Pause/Resume functionality with a single toggle button.  
- Copy quote and author to clipboard with a dedicated button.  
- Smooth fade animations for quote transitions.  
- Fully responsive and lightweight design.  
- Clean, modern UI with icon-based controls (Font Awesome).  

---

## How it Works

1. The app fetches a random quote from the API.  
2. The reading timer is automatically calculated based on the number of words in the quote.  
3. The progress bar visually represents the reading duration.  
4. Users can pause or resume the timer using the toggle button.  
5. Users can copy the current quote with the copy button.
